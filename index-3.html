Patricia Loren

Search Engines with Python and Elasticsearch

A search engine is a software responsible for search into large amounts of data. Often, involved algorithms are complex and their architecture are not trivial. In this scenario, Elasticsearch appears as a great solution to solve the indexing and searching tasks. In this talk I will show how ElasticSearch and Python can make the development of search engines more easier. Also, I will show how Python can be used to build Web Crawlers and Scrapers and how to send gathered data from the web to be indexed in Elasticsearch.

Programming to the extreme

This talk is about sharing how a day of programming to extreme looks like, what are the struggles and mostly what are the benefits of writing code by starting with the tests (TDD), push code directly to production (covering CI, CD concepts) and last, but not least doing this with a pair side by side, 8 hours a day, every day. This methodology, even though had been around for some time now, it’s still a tabu subject and it’s still seen as a hard to approach process. I would like, via this talk to solve some of the doubts and uncertainties that lie around this subject.

Berta Devant

The fake problem with hiring juniors

There is a common misconception that junior developers are a burden and won’t be delivering value for the first months or even years. Yet, observing my own contributions as a junior, I realised how my insights were useful for my team from day one. Working with agile work methodologies, using pair programming and code review, you can contribute to your team, deliver value and learn on the job at the same time In this session, we will discuss these misconceptions and how in reality a good developer, no matter their experience level, can bring real value to your team and business. Then we will address the real question: ""You hired a good dev who is a junior, how do you train them properly so they can grow?"" Three takeaways for the audience: 1. Understand the value juniors can bring 2. Learn ways of enhancing junior inclusion to help them grow 3. How to break away from imposter syndrome
